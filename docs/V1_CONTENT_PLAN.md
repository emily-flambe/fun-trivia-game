# V1 Content Plan — Trivia Trainer

Complete content target for V1, organized by Learned League's 18 categories. Informed by [TrivialStudies.com](https://www.trivialstudies.com/) coverage and LL question patterns.

**Philosophy**: Omnibus exercises that cover a topic broadly. Specialized sub-exercises (by era, by sport, by genre) come later as V2 refinements.

**Status legend**:
- ✅ **Done** — exercise exists in production
- 📋 **Ticketed** — Linear ticket exists (may be done or in backlog)
- ❌ **New** — needs a Linear ticket
